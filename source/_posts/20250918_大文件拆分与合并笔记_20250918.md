## 大文件拆分与合并笔记

由于光盘容量有限，如果需要拷贝 超过单个光盘大小的大文件，可以先将其 拆分成多个小文件，分别拷贝，再在目标机器上 合并解压。

方法：边压缩边拆分

使用 tar + gzip + split，可以在压缩的同时完成拆分：

拆分命令

```shell
split -b 2600M hsw_crawl4ai_0918.tar hsw_crawl4ai_0918.tar.part-

```


这里：

tar -czf - ：将 hsw_crawl4ai_0918.tar 压缩并输出到标准输出（stdout）

split -b 2600M - ... ：每 2.6GB 拆分为一个小文件

输出文件前缀为 hsw_crawl4ai_0918.tar.gz.part-

生成的文件示例：

```
hsw_crawl4ai_0918.tar.gz.part-aa
hsw_crawl4ai_0918.tar.gz.part-ab
hsw_crawl4ai_0918.tar.gz.part-ac
...
```

合并与解压

在目标机器上，先合并，再解压：

```shell
cat hsw_crawl4ai_0918.tar.gz.part-* > hsw_crawl4ai_0918.tar.gz
tar -xzf hsw_crawl4ai_0918.tar.gz
```


这样就能还原出原始文件内容。
