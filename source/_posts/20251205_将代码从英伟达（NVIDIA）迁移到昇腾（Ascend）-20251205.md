# 将代码从英伟达（NVIDIA）迁移到昇腾（Ascend）

## 一、自动迁移方式

在 PyTorch 代码中，引入昇腾 NPU 相关库，并使用自动迁移工具：

```python
import torch
import torch_npu
from torch_npu.contrib import transfer_to_npu
```

`transfer_to_npu` 可用于将原本基于 CUDA 的代码自动迁移到 NPU 运行环境，减少手动修改成本。

## 二、指定使用的昇腾卡

如需指定使用某一张昇腾卡，需要**在程序最开始**设置环境变量：

```python
import os
os.environ["ASCEND_RT_VISIBLE_DEVICES"] = "7"
```

说明：

- 该语句必须写在 **所有 `torch` / `torch_npu` 相关 import 之前**
- `"7"` 表示使用第 8 张昇腾卡（从 0 开始编号）
- 多卡可使用逗号分隔，例如：`"0,1,2"`
